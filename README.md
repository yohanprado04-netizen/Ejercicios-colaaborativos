# Ejercicios-colaaborativos
mi ejercicio es sobre un servicio de streaming, en este ejercicio cree una clase padre y dos subclases que ambas heredaron el metodo reproducir, pero cada una de las clases hijas tienen su propio comportamiento, ademas utilice un un ciclo para crear un menu con tres opciones disponibles para el usuario, donde se le pregunta al usuario por el servicio que quiere consumir y el elije la opcion que le convenga. utilice los condicioneales if, elif, else, para validar que el usuario elija solo las opciones disponibles y en cada opcion utilice el obejeto que cree y llame la funcion reproducir, para que de acorde a la opcion que elija el usuario, haga el comportamiento que le corresponte.

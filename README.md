# Ejercicios-colaaborativos
El primer ejercicio trata sobre el procesamiento de pagos personalizados segun los metodos de pago disponoibles, tales como Efectivo, Tarjeta, Transferencia.
Esto con el uso de clases, herencia y polimorfismo para personalizar los resultados y enraizandolos a la clase padre.


El segundo ejercicio:

Se enfoca en la reserva cupos para viaje en: Lancha y Bus.

En este caso, se utliliza el polimorfismo y se inicializa el metodo super para utilizar la cantidad de pasajeros heredadad deste la clase reserva, quien es padre
a sus hijas, quienes son las reserva en los tipos disponibles tales como, Lancha y Bus.
